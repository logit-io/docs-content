Filebeat is a lightweight shipper that enables you to send your 
Docker container application logs to your Logit.io Stack. 
Configure [Filebeat](/integrations/log-management/shippers/filebeat) using the pre-defined 
examples below to start sending and analysing your Docker application logs.

<Steps>
  ### Install Integration
  <InstallIntegration/>
  ### Install Filebeat

  <InstallFilebeat />

  ### Configure Filebeat

  <FilebeatContainer>
    ```yaml copy
    # ============================== Filebeat inputs ===============================
    filebeat.inputs:

    - type: container

      enabled: true

      paths:
      - 'var/lib/docker/containers/*/*.log'

    # ================================== Outputs ===================================
    output.logstash:
        hosts: ["@logstash.host:@logstash.sslPort"]
        loadbalance: true
        ssl.enabled: true
    ```
  </FilebeatContainer>

  ### Validate configuration

  <ValidateBeat beatname="filebeat" />

  ### Start filebeat
  
  <StartFilebeat />

  ### Launch Logs to View Your Data

  <LaunchVisualizer type="logs" />


  ### How to diagnose no data in Stack

  <DiagnoseNoData />
