Use OpenTelemetry to easily send Node.js traces to your Logit.io Stack.

<Steps>
  ### Getting Started

  You can get help Getting Started with OpenTelemetry for Node.js by referring to the 
  [official documentation](https://opentelemetry.io/docs/instrumentation/js/getting-started/). 
  
  You need to configure the OTLP exporter to ensure the trace data reaches your Logit.io Stack.
  
  OTLP gRPC Exporter settings:

  ```js copy showLineNumbers /@opentelemetry.protocol/ /@opentelemetry.endpointAddress/ /@opentelemetry.port/ /@opentelemetry.base64encodedAuthString/
  const { OTLPTraceExporter } =  require('@opentelemetry/exporter-trace-otlp-@opentelemetry.protocol');

  const collectorOptions = {
    url: 'https://@opentelemetry.endpointAddress:@opentelemetry.port',
    headers: {'Authorization': 'Basic @opentelemetry.base64encodedAuthString'}
  };
  ```

  For more information about exporter configuration with Node.js 
  please refer to the [documentation](https://opentelemetry.io/docs/instrumentation/js/exporters/).
