# What versions of the Logging Stack does Logit.io support?

The Logit.io platform enables you to manage multiple isolated ELK versions across your infrastructure.

**OpenSearch:** All versions of OpenSearch are supported, including the latest OpenSearch Stack releases.

**OpenSearch Dashboards:** All versions of OpenSearch Dashboards are supported.

**Logstash:** All versions of Logstash are supported.

**ELK:** The last supported ELK versions are:
- Elasticsearch 7.10, Logstash 7.10, Kibana version 7.10, Opendistro 1.13
- Elasticsearch 7.8, Logstash 7.8, Kibana version 7.8
