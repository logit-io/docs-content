# Managing Stacks via API

Automate stack management using the Logit.io Developer API. This guide covers listing stacks, retrieving connection details, and requesting upgrades.

## List Your Stacks

First, get your account ID from `/api/accounts`, then fetch stacks for that account:

<Tabs items={["Linux / macOS", "Windows"]}>
<Tab>

```bash copy /@dashboardApiKey/
# Get accounts
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/accounts

# Get stacks for an account (replace ACCOUNT_ID with your account ID)
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/account/ACCOUNT_ID/stacks
```
</Tab>
<Tab>

```cmd copy /@dashboardApiKey/
# Get accounts
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/accounts

# Get stacks for an account (replace ACCOUNT_ID with your account ID)
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/account/ACCOUNT_ID/stacks
```
</Tab>
</Tabs>

The response returns subscriptions and their associated stacks, including stack IDs and product names.

</Tab>
<Tab>

```cmd copy /@dashboardApiKey/
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/stacks/STACK_ID
```

Replace `STACK_ID` with your stack's GUID. The response includes stack name, type, and status.

## Request a Stack Upgrade

For Log Management stacks, you can check available versions and request upgrades:

<Tabs items={["Linux / macOS", "Windows"]}>
<Tab>

```bash copy /@dashboardApiKey/
# Check upgrade options
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/stacks/STACK_ID/upgrade

# Request upgrade (POST with versionId in body)
curl -X POST -H "x-api-key: @dashboardApiKey" -H "Content-Type: application/json" \
  -d '{"versionId": "VERSION_ID"}' \
  https://dashboard.logit.io/api/stacks/STACK_ID/upgrade/requests
```
</Tab>
<Tab>

```cmd copy /@dashboardApiKey/
# Check upgrade options
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/stacks/STACK_ID/upgrade

# Request upgrade (POST with versionId in body)
curl -X POST -H "x-api-key: @dashboardApiKey" -H "Content-Type: application/json" ^
  -d "{\"versionId\": \"VERSION_ID\"}" ^
  https://dashboard.logit.io/api/stacks/STACK_ID/upgrade/requests
```
</Tab>
</Tabs>

For more on upgrades, see [Migrate or Upgrade Your Stack](/account-administration/subscriptions-management-usage/product/migrate-upgrade-your-stack).

## Audit Log

Search the audit log for stack activity:

<Tabs items={["Linux / macOS", "Windows"]}>
<Tab>

```bash copy /@dashboardApiKey/
curl -H "x-api-key: @dashboardApiKey" "https://dashboard.logit.io/api/stacks/STACK_ID/audit-log?Page=1&PageSize=20"
```
</Tab>
<Tab>

```cmd copy /@dashboardApiKey/
curl -H "x-api-key: @dashboardApiKey" "https://dashboard.logit.io/api/stacks/STACK_ID/audit-log?Page=1&PageSize=20"
```
</Tab>
</Tabs>

Optional query parameters: `Search`, `Period`.
