# Getting Started

This quickstart walks you through making your first request to the Logit.io Developer API.

## Prerequisites

- A [Logit.io account](https://dashboard.logit.io/sign-up)
- An API key from your [Profile](https://dashboard.logit.io/profile)

## Step 1: Create an API Key

1. Log in to the [Logit.io Dashboard](https://dashboard.logit.io).
2. Go to **Profile** and find the **API Keys** section.
3. Create a new API key and copy it (it is only shown once).

## Step 2: Make Your First Request

Verify your setup by fetching your profile:

<Tabs items={["Linux / macOS", "Windows"]}>
<Tab>

```bash copy /@dashboardApiKey/
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/me/profile
```

</Tab>
<Tab>

```cmd copy /@dashboardApiKey/
curl -H "x-api-key: @dashboardApiKey" https://dashboard.logit.io/api/me/profile
```

</Tab>
</Tabs>

A successful response returns your user profile and a summary of your API keys.
